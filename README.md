## 👤 关于我
习惯在纷杂里找一条安静的路。  
读经典，不为猎奇，只为把一件事看得更深一点。  
写博客，不为热闹，只为把思路整理得更清楚一点。  

在传统医学与数字世界之间行走：  
一手翻书，一手敲键盘。  
用代码搭建秩序，用文字沉淀理解。  

相信长期主义，相信慢慢来会比较快。  
愿所学能自用，所思能自明。
